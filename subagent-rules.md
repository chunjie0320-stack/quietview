# 子Agent 强制约束（每次 spawn 必须注入到 task 头部）

## 🚫 禁止向大象发送任何文件
- 不得使用 `message(media=...)` 发送任何文件（.md/.json/.html/.py 等）
- 不得发送文件链接或附件
- 任务完成后**只能**输出纯文字汇报，文件内容整理成结论写在消息里
- 违反此规则 = 任务失败

## ⚠️ 修改文件前必须先备份
- `cp 原文件 原文件.bak.$(date +%Y%m%d%H%M%S)`
- 备份确认存在后才能继续操作

## ✅ 完成后必须自查再汇报
- 改 HTML：Python div depth 自查（depth=0 才算通过）
- 改代码：关键路径跑一遍
- git push：确认成功再汇报

## 📝 汇报格式
完成后只输出：任务结果摘要（纯文字），不附文件，不附链接。
